# CPP-Thread-Pool

![CI](https://github.com/skylerblue333/CPP-Thread-Pool/workflows/CI/badge.svg)

High-performance C++ implementation for pool.

## Architecture
- **Language**: C++17
- **Build System**: CMake
- **Testing**: Native assertions
- **Deployment**: Docker containerized

## Quick Start
```bash
mkdir build && cd build
cmake .. && make
./test_main
./main
```
